# John Doe Portfolio

A production-ready personal portfolio website for John Doe.

## Features

- **Built with**: Vite, React, TypeScript.
- **Routing**: `react-router-dom` (SPA).
- **Styling**: Plain CSS (Minimalist & Professional).
- **Deployment**: Automated via GitHub Actions to GitHub Pages.
- **Data-Driven**: Content is managed in `src/data/`.

## Getting Started

### Prerequisites

- Node.js (v20+)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment

Deployment is automated via GitHub Actions on every push to the `main` branch. 
The site is hosted at `https://glacode.github.io/portfolio/`.
